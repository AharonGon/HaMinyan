# המניין

אפליקציית אנדרואיד לאיתור מנייני תפילה בקרבתכם, מבוססת על מאגר המניינים של נדרים פלוס.

## הורדה

הגרסה האחרונה זמינה להורדה בעמוד
[GitHub Releases](https://github.com/AharonGon/HaMinyan/releases/latest).

האפליקציה בודקת אוטומטית בעת ההפעלה אם פורסם Release חדש. ניתן גם לבצע
בדיקה ידנית דרך **הגדרות > בדיקת עדכון**.

## מה יש באפליקציה

- **בקרבתי** - איתור מניינים לפי מיקום המכשיר (באישור המשתמש), עם בחירת רדיוס חיפוש
- **חיפוש** - חיפוש בית כנסת לפי שם, עם היסטוריית חיפושים אחרונים
- **פרטי בית כנסת** - לוח זמנים מלא לכל ימות השבוע, מקובץ לפי סוג תפילה
- **מועדפים** - שמירת בתי כנסת לגישה מהירה

## פיצ'רים לשיפור חוויית המשתמש

1. **ספירה לאחור למניין** - כל מניין שטרם התקיים היום מציג "בעוד X דק׳"
2. **הדגשת המניין הבא** - במסך פרטי בית הכנסת, המניין הקרוב ביותר מודגש בצבע
3. **סינון לפי סוג תפילה** - צ'יפים לסינון שחרית / מנחה / ערבית / שיעור
4. **בחירת רדיוס חיפוש** - 1 עד 15 ק"מ, נשמר כהעדפה
5. **ניווט בלחיצה** - פתיחת Waze / Google Maps ישירות לבית הכנסת
6. **שיתוף לוח זמנים** - שליחת זמני התפילות של בית כנסת בוואטסאפ או בכל אפליקציה
7. **תאריך עברי** - הצגת התאריך העברי והיום בשבוע במסך הראשי
8. **תצוגת לוח לפי יום** - בחירת יום בשבוע לצפייה בלוח הזמנים המתאים
9. **מצב כהה / בהיר / אוטומטי** - ערכת נושא לבחירה בהגדרות
10. **משיכה לרענון** - רענון רשימת המניינים בתנועת משיכה, כולל חותמת "עודכן ב-"
11. **עדכונים אוטומטיים** - בדיקה מול GitHub Releases והורדת APK חדש

## מבנה טכני

- Kotlin + Jetpack Compose (Material 3), תמיכת RTL מלאה
- MVVM: Retrofit + OkHttp מול ה-API של נדרים פלוס, DataStore להעדפות ומועדפים
- Fused Location Provider לאיתור מיקום
- אייקונים בלבד מ-Material Icons (ללא אימוג'ים)
- minSdk 26 (אנדרואיד 8.0), targetSdk 35

## בנייה והתקנה

דרישות: JDK 17 ו-Android SDK (מותקנים אוטומטית עם Android Studio).

```powershell
# בניית APK לבדיקה
.\gradlew.bat assembleDebug

# הקובץ נוצר כאן:
# app\build\outputs\apk\debug\app-debug.apk

# בניית APK חתום להפצה (דורש keystore.properties מקומי)
.\gradlew.bat assembleRelease
```

להתקנה על מכשיר המחובר ב-USB (עם ניפוי USB פעיל):

```powershell
adb install app\build\outputs\apk\debug\app-debug.apk
```

או פתחו את התיקייה ב-Android Studio והריצו על מכשיר/אמולטור.

## הערות

- הנתונים מגיעים בזמן אמת משרתי נדרים פלוס (matara.pro) - נדרש חיבור לאינטרנט
- זמני התפילות באחריות בתי הכנסת המפרסמים במאגר
- מפתח החתימה וקובץ `keystore.properties` אינם נשמרים במאגר. יש לגבות אותם
  במקום מאובטח; בלעדיהם לא ניתן לפרסם עדכון שניתן להתקין מעל גרסה קיימת.
